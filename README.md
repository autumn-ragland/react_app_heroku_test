# React App to Test Heroku Deployment 

### React Deployment Steps
1. Add the code snippet below to the `package.json` file
```JS
"engines": {
    "npm": "6.14.5",
    "node": "12.18.1"
  }
```
2. Run `heroku create APPNAME --buildpack mars/create-react-app` in the react app directory
